# 리뷰

## 클래스와 생성자 함수의 차이

## this 바인딩

| 함수 호출 방식       | this가 가리키는 값(this 바인딩)        |
| -------------------- | -------------------------------------- |
| 일반 함수로서 호출   | 전역 객체                              |
| 메서드로서 호출      | 메서드를 호출한 객체(마침표 앞에 객체) |
| 생성자 함수로서 호출 | 생성자 함수가 (미래에) 생성할 인스턴스 |

## 생성자 함수 암묵적 처리

- this를 빈 객체({})로 초기화
- return this
